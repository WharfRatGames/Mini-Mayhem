# Mini-Mayhem Wiki

**Mini-Mayhem** (code name *Arty*) is a Worms-style artillery game for the **Miyoo Mini Plus** handheld, written entirely in Rust. Two teams of soldiers lob explosives at each other across destructible terrain until only one side is standing.

## Quick Start

1. Download the latest build from the [releases page](http://crumbonium.duckdns.org/arty/builds/) or install via OTA update
2. Launch **Arty** from the Miyoo home screen
3. Pick a mode from the title menu — **HOTSEAT** for local play, **LIVE GAME** or **TAKE A TURN** for online
