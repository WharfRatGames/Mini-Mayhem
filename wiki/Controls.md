# Controls

## Aiming & Movement

| Button | Action |
|---|---|
| **Left / Right** | Move soldier |
| **Up / Down** | Aim up / down |
| **A** | Fire / confirm |
| **B** | Jump / cancel / back |
| **X** | Open weapon menu |
| **Y** | Toggle camera follow |
| **L1** | Decrease fuse timer (Grenade / TNT) |
| **R1** | Increase fuse timer / Camera pan hold |
| **R1 + D-pad** | Free camera pan |
| **Start** | Pause / end turn early |
| **Select** | Open weapon menu (alternate) |

## Weapon Menu

| Button | Action |
|---|---|
| **Up / Down / Left / Right** | Navigate weapons |
| **A** | Select weapon |
| **B / Select** | Close menu |
| **L1 / R1** | Adjust fuse (Grenade only — TNT fuse is automatic) |

## Ninja Rope

| Button | Action |
|---|---|
| **A** | Fire hook |
| **Up / Down** | Shorten / lengthen rope |
| **Left / Right** | Swing |
| **B** | Release rope |

## Plasma Torch

| Button | Action |
|---|---|
| **Up** | Torch up-forward |
| **Right/Left** | Torch forward (facing direction) |
| **Down** | Torch down-forward |
| **B** | Stop torching |

## Title Screen / Menus

| Button | Action |
|---|---|
| **Up / Down** | Navigate |
| **A** | Confirm |
| **B** | Back |
