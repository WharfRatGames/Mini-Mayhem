Mini Mayhem / Arty — Progress Summary
Last updated: 2026-06-22 (v0.5.4.328)

=== Phase 1 — Terrain & Physics: DONE ===
Perlin heightmap, crater carving, Euler projectile ballistics, wind, water.

=== Phase 2 — Miyoo Renderer & Input: DONE ===
/dev/fb0 BGRA framebuffer, evdev buttons, 8x8 font, hot-seat local play.

=== Phase 3 — Weapons & Turn System: DONE ===
18 weapons implemented: Bazooka, Grenade, Shotgun, TNT, Landmine, Meteor Bomb,
Baseball Bat, Revolver, Ninja Rope, Blasthive, Black Hole Bomb, Plasma Torch,
Hand of Jerry (Garcia), Air Strike, Minigun, Mac-10, Holy Hand Grenade, Homing Missile.
Tiered crate drops (Common/Uncommon/Rare/Ultra Rare). Shared team loadout, turn
locks, weapon menu. Graves, crates, barrels, fire patches, black holes all synced.

=== Phase 4 — Dedicated Server (RPi4): ~95% ===
Live multiplayer works. Take-a-Turn async works. Ranked matchmaking built into
the game server (single port 7777, all match types as threads). Cross-compiled
aarch64 server binary. All multiplayer traffic encrypted with TLS/SSL (TLS 1.2/1.3
via rustls + Let's Encrypt cert). DNS pre-resolved at game launch so live connect
is instant (no 15-20s delay). External connectivity confirmed: port 7777 open,
DuckDNS current, no firewall issues, TLS cert valid until Sep 2026 with auto-renewal.

Build commands (as of v0.5.4.328):
  cargo miyoo --bin mini-mayhem          (Miyoo armv7 client)
  cargo piserver --bin server            (Pi aarch64 server, --no-default-features)
  cargo build --release --target x86_64-pc-windows-gnu --bin mini-mayhem  (Windows, release ONLY)
  bash deploy/update_server.sh <ver>     (deploys all three + Miyoo push + GitHub release)
